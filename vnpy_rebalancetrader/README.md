## 说明

RebalanceTrader是用于组合调仓交易委托执行的模块。


## 安装

安装环境推荐基于3.0.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。


下载源代码后，解压后在cmd中运行：

```
pip install .
```
