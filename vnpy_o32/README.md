## 说明

基于恒生O32的API接口封装开发，只支持Python 3.7版本。

## 安装

安装需要基于3.0.0版本以上的[VeighNa Studio](https://www.vnpy.com)。

直接使用pip命令：

```
pip install vnpy_o32
```


或者下载解压后在cmd中运行：

```
python setup.py install
```
